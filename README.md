# Gestor de dispositivos personales
Este proyecto contempla el desarrollo de una aplicacion en la que tener acceso a los datos y la configuracion de los diferentes dispositivos que se pueden tener en casa o en cualquier otro sitio. Por ejemplo, desde esta app puedes tener accceso a los datos de tus sensores de temperatura de cada habitacion, tener un mapa de casa o de la nave donde visualizar la informacion y posicion de cada sensor o actuador o bien saber el estado de tu alarma o de tus puertas o ventanas.

El objetivo principal es centralizar toda la informacion de los dispositivos en un solo lugar y que tenga soporte para el mayor numero de dispositivos.
